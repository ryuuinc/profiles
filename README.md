# profiles

[![Build Status](https://github.com/ryuuinc/profiles/workflows/Update/badge.svg)](https://github.com/ryuuinc/profiles/actions)

## 介绍

每天 10 点自动更新用于 `Clash`、`Quantumult`、`Shadowrocket` 等软件的规则碎片和一些工具
